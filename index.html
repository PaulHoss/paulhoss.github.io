About Me

I’m Paul Hoss, an Analytics Engineer focused on building automated data pipelines, cloud‑native data models, and reliable analytics systems. I work primarily with Python, SQL, AWS, and dbt, and I enjoy turning messy, unstructured data into clean, trustworthy datasets and dashboards.

My featured project, the RAM Market Monitor, reflects how I approach engineering: I focus on designing data workflows that are reliable, automated, and easy to maintain as they scale. I’m especially interested in creating systems that stay simple and transparent, even as the data grows more complex.

I’m actively expanding my skills and looking for opportunities to contribute to data engineering, analytics engineering, or data‑driven product teams.

Projects

Memory Market Monitor

An end-to-end analytics engineering pipeline that uses Python to scrape daily RAM prices from e-commerce retailers, archives raw data to an S3 Bronze data lake, transforms it through Silver and Gold layers using dbt over AWS Athena, and publishes an interactive price-tracking dashboard to GitHub Pages via Evidence. Fully automated with GitHub Actions on a daily schedule.
